## 1. 产品概述
SmartLearner 是一款面向大学生的个性化学习与知识管理 Agent，以 RAG（检索增强生成）为核心架构，实现知识的主动获取、深度消化与高效复用。目标用户为合肥工业大学在校生，通过 CAS 统一认证直连教务系统，构建"以我为中心"的智能知识工作流。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学生 | 合工大CAS统一认证登录 | 课程管理、知识库、论文精读、学习规划、团队协作 |
| 教师 | CAS认证 + 角色升级 | 课程管理、发布公告、查看学习数据 |

### 2.2 功能模块
1. **智能仪表盘**：学习概览、效率评分、连续打卡、今日进度、最近活动
2. **课程问答助手**：上传课件/教材，构建可对话、可测验的智能知识库
3. **论文精读教练**：自动总结、解答、提问，推荐关联文献
4. **个人知识库管家**：自动归档收藏内容，自然语言精准秒查
5. **学习路径规划师**：根据目标定制计划、推送资料、跟踪进度
6. **团队协作学习**：小组共享待办与公告
7. **智能日报统计**：数据驱动的学习分析

## 3. 核心流程

### 3.1 用户登录与课表获取
用户通过CAS认证登录 → 系统获取学号信息 → 自动拉取课表 → 展示仪表盘

### 3.2 课程问答流程
选择课程 → 上传课件 → 系统自动解析分块索引 → 用户提问 → RAG检索+生成回答

### 3.3 论文精读流程
上传论文 → AI自动分析（摘要/方法/发现/贡献） → 用户可追问 → 推荐关联文献

### 3.4 知识库管理流程
添加知识条目 → 自动向量化索引 → 自然语言搜索/问答 → 归档管理

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝 (#1E3A5F) + 翡翠绿 (#10B981) 作为活力点缀
- **辅助色**：暖灰 (#F8FAFC) 背景，深灰 (#1E293B) 文字
- **按钮风格**：圆角(8px)，主按钮深蓝实色，次按钮描边
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局
- **图标**：Lucide React 图标库，线性风格
- **动效**：页面切换淡入，卡片悬浮微动，数据加载骨架屏

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | CAS认证 | 居中卡片，合工大校徽，登录按钮，渐变背景 |
| 仪表盘 | 学习概览 | 4个统计卡片(课程/知识/今日时长/效率)，7日趋势折线图，连续打卡火焰图标 |
| 仪表盘 | 最近活动 | 时间线列表，带图标的活动条目 |
| 课程页 | 课程列表 | 网格卡片，课程名/教师/学期，收藏星标 |
| 课程页 | 课程问答 | 左侧对话气泡，右侧引用文档面板 |
| 课程页 | 智能测验 | 题目卡片，选项按钮，解析折叠 |
| 论文页 | 深度分析 | 信息卡片组(研究问题/方法/发现/贡献)，标签式切换 |
| 知识库页 | 语义搜索 | 大搜索框，搜索结果卡片列表，分类侧边栏 |
| 学习规划页 | AI规划 | 阶段时间线，里程碑节点，进度条 |
| 团队页 | 共享待办 | 待办列表，勾选框，负责人头像，截止日期 |

### 4.3 响应式设计
- 桌面优先设计，最小宽度 1024px
- 平板适配：导航栏折叠为汉堡菜单
- 移动端：单列布局，底部导航

### 4.4 特色交互
- 课表周视图：时间轴 + 课程色块，点击查看详情
- 知识搜索：输入即搜，实时展示结果
- 对话界面：打字机效果，引用来源可点击
- 效率评分：环形进度条动画
